# mm-crypt-cli

Command-line interface and TUI editor built on top of [mm-crypt](../mm-crypt/README.md).

> **Status:** Early scaffolding — placeholder `hello` command only.

## Install

```bash
uv tool install mm-crypt-cli
```

This installs the `mm-crypt` executable:

```bash
mm-crypt hello
```

## Commands

- `mm-crypt hello [NAME]` — placeholder command used to verify the wiring.
